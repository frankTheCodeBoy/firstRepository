---

### 📄 File Overview

- `hello_git.py`:  
  A simple script that prints a greeting message. Ideal for testing Git commits, pushes, and repository setup.

---

### 🚀 Getting Started

To run the script:

```bash
git clone https://github.com/frankTheCodeBoy/firstRepository.git
cd firstRepository
python hello_git.py
```
